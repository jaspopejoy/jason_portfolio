Details

AI Chef is powered by OpenAI. This app allows the user to add ingredients they hava available and will return multiple recipes. These recipes include an image and the ability to print the recipe.

This app was built using Blazor web Assembly, ASP.NET, C#, and some JavaScript. It includes the use of OpenAI API, ChatGPT, DALL-E, and REST API
